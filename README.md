# Digital Bookshelf API

A simple RESTful backend for managing a library's book inventory.  
Built with **Express**, **MongoDB**, and **Mongoose**.

## Features

- Create new book records
- View all books
- Retrieve a book by ID
- Update a book by ID
- Delete a book by ID
- Mongoose models with validation
- Modular Express routes
